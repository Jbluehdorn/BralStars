# Adventurers' Board

*The corkboard at [[The Laughing Beholder]]. Anyone can post; anyone can pick up a job.*

> [!note] How Jobs Work
> Any crew can pull a job off the board. Post in Discord when you want to run one — form a party, name the job, pick a date. See the [[West Marches Player Guide]] for the full format.
